## 🎮 What is TCG Sandbox?

TCG Sandbox is a platform that lets you:

- **Create** custom trading card games with your own rules, cards, and sets
- **Build** decks from available cards in any game
- **Play** in real-time multiplayer lobbies with other players
- **Share** your creations with the community

Currently, the platform focuses on game creation and gameplay mechanics. While traditional TCGs involve card ownership and trading, TCG Sandbox prioritizes the creative and competitive aspects of the genre.

## ℹ️ About This Project

TCG Sandbox is a closed-source passion project developed and maintained by an individual software engineer who loves trading card games. While the codebase is not *yet* open source, community feedback and suggestions are welcomed and valued.

### How Feedback Works

1. **You submit**: Share bugs, ideas, questions, and suggestions through GitHub issues
2. **I review**: All submissions are read and considered
3. **I decide**: Final decisions about implementation and priority rest with the maintainer
4. **I communicate**: Issue statuses are updated and context is provided when possible

Please understand that:
- Not all suggestions can be implemented
- Detailed explanations may not be provided for every decision
- Response times may vary based on project priorities

This approach helps maintain development momentum while staying connected with the community.
