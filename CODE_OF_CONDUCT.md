# Code of Conduct

## Our Commitment

TCG Sandbox is committed to providing a welcoming and inclusive environment for everyone interested in trading card games, game design, and software development.

## Expected Behavior

When participating in the TCG Sandbox community, we expect you to:

- **Be respectful**: Treat others with courtesy and respect, even when you disagree
- **Be constructive**: Provide helpful feedback and criticism that moves conversations forward
- **Be patient**: Remember that not all suggestions can be implemented immediately (or at all)
- **Be understanding**: This is a solo-developed passion project with limited resources
- **Stay on topic**: Keep discussions relevant to TCG Sandbox features, bugs, and improvements

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, intimidation, or discrimination of any kind
- Personal attacks or insults
- Spam, advertising, or off-topic promotion
- Sharing others' private information without permission
- Repeatedly demanding features or responses after being declined

## Scope

This Code of Conduct applies to all community spaces, including:

- GitHub issues and discussions
- Project documentation
- Any other official TCG Sandbox communication channels

## Enforcement

Violations of this Code of Conduct may result in:

1. A warning
2. Temporary or permanent ban from community spaces
3. Removal of content that violates these guidelines

The project maintainer has final discretion on enforcement decisions.

## Contact

If you experience or witness unacceptable behavior, or have concerns about conduct, please contact the maintainer directly through GitHub.

## Attribution

This Code of Conduct is adapted from established open source community guidelines and tailored for TCG Sandbox's community needs.
